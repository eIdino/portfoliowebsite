Projects

Plugin for user registration and course creation

I created a Moodle plugin in PHP to automate the process of creating a course by reading values from a .csv file and register the students into the created course. I also added extra functionality such as extending the lifetime of the course if it reaches the end date and checking for duplicates in the course.

Server-side and Client-side web app

3-person group project that consisted of creating an online store selling PC components. The web page was created using Python Django/Wagtail CMS and designed using Bootstrap and CSS. Some functionality included a virtual shopping cart for storing items, a voucher code system for discounts, sign up and sign in system for creating and logging into accounts as well as 2 Factor Authentication (2FA) for extra security. Credit card payment system using Stripe and a live support chat using a WebSocket. The stores database was managed using SQLite.

Client-side web app

My first ever project, creating a simple marketplace on the web using Python Django that sells furniture. Contained a search bar for searching specific furniture, shopping cart, and a credit card system.
